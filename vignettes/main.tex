\hypertarget{outline}{%
\section{Outline}\label{outline}}

The app data is contained in 2 main scripts, \texttt{params.R} and
\texttt{functions.R}. The former consists of the main functions managing
the app, and the latter defines starting parameters and user options.

\centering

\hypertarget{running-the-app}{%
\section{Running the app}\label{running-the-app}}

Currently, the app works within a standard R package format, with the
exception of \texttt{app.R}, which lives in the main directory. To run
the app, either run \texttt{app.R} from an Rsession or from command line
with:

\begin{Shaded}
\begin{Highlighting}[]
\ExtensionTok{Rscript}\NormalTok{ ./app.R}
\end{Highlighting}
\end{Shaded}
